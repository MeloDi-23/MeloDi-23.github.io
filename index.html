新手记录注意事项

git bash的使用命令

git环境配置

git config --global user.email "<email address>"
git config --global user.name "<name>"
git init
添加工作点
git add .
git remote add origin <github keygen, use ssh>
本地提交
git commit -m"<message>"
上传
git push/pull origin <branch name, ussually is master>
如果出现冲突错误，以云端为准，先pull，修改完成后再push。养成习惯尽量用bash，少用网页操作！
